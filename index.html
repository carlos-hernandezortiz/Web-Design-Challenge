Summary : Latitude vs. X

The purpose of this project was to analyze how weather changes as you get close to equator. To accomplish the analysis, we first pulled the data from OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. lattitude. Factors we looked at included temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as a part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
